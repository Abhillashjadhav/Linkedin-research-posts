# LinkedIn Authority OS

Recovered and upgraded content workflow for Abhillash Jadhav.

## Purpose
Build GenAI product authority that converts into speaking, podcast, advisory, project, and senior product opportunities.

## Pipeline
Scout → Analyst → Writer → Critic → Human approval → Publish → 24h/72h review.

## Daily use
1. Clone the repository.
2. Check out `recovery/linkedin-authority-os-v6`.
3. Open it in Claude Code.
4. Run `/draft-post` or `/draft-post <topic> --goal authority`.
5. Review the winning draft and publish manually.

The system does not auto-publish. Human approval is mandatory.
